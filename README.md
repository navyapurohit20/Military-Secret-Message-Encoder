🔐 Military Secret Message Encoder

📌 Project Overview
The **Military Secret Message Encoder** is a Python-based application designed to provide secure communication through encryption and decryption of messages. It demonstrates basic cryptographic techniques by converting readable text (plaintext) into an unreadable format (ciphertext) and vice versa.

This project is built as part of a Machine Learning / Security Lab to help understand fundamental concepts of **cryptography, data security, and GUI-based applications**.

 🚀 Features
- 🔒 Encrypt messages using a two-step encryption method  
- 🔓 Decrypt messages back to original form  
- 🧠 Uses Caesar Cipher (+2 shift)  
- 🔁 Reverse text encryption technique  
- 🖥️ User-friendly GUI using Tkinter  
- 📋 Copy output to clipboard  
- 💾 Save encrypted message to a file  
- 🧹 Clear input/output fields  
- ⏰ Live system clock display  


 Technologies Used
- **Python** (Core programming language)  
- **Tkinter** (GUI development)  
- **Cryptography Concept** (Caesar Cipher)  
- **File Handling** (Save messages)  
- **Time Module** (Live clock feature)  



 How It Works

 Encryption Process
1. User enters a message  
2. Message is **reversed**  
3. Each letter is shifted **+2 positions** in the alphabet  
4. Encrypted message is displayed  

Decryption Process
1. Input encrypted message  
2. Letters are shifted **-2 positions**  
3. Message is reversed again  
4. Original message is obtained
